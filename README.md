# Product Analytics & Experiment Framework

This project demonstrates an end-to-end product analytics workflow:
- Raw event tracking
- SQL transformations
- Retention & funnel analysis
- A/B test simulation
- Product decision memo

## Stack
- SQL
- Python (pandas, statsmodels)
- Jupyter

## Project Question
Does a redesigned onboarding flow improve 7-day user retention?

## How to Run
1. Install dependencies
2. Generate or load event data
3. Run notebooks in order
